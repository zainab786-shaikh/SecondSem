Get Coding



Author: Young Rewired State
Date of publish: 30 June 2016

ABOUT:
Young Rewired State is a worldwide community of digital makers, coders and geeks aged 18 and under. The community allows young coders to meet like-minded peers and share their skills. Young Rewired State members can also meet expert mentors at free events around the world, like the Festival of Code, and use freely available open data to build apps, websites and algorithms to solve real world challenges. So join Young Rewired State today and become a technology star of the future! Learn how to write code and then build your own website, app and game using HTML, CSS and JavaScript in this essential coding guide for kids from expert organization Young Rewired State.Learn how to write code and then build your own website, app and game using HTML, CSS and JavaScript in this essential guide to coding for kids from expert organization Young Rewired State. Over 6 fun missions learn the basic concepts of coding or computer programming and help Professor Bairstone and Dr Day keep the Monk Diamond safe from dangerous jewel thieves. In bite-size chunks learn important real-life coding skills and become a technology star of the future. Young Rewired State is a global community that aims to get kids coding and turn them into the technology stars of the future.

SUMMARY:
"The Get Coding! books are essential guides to programming for kids. The fun series is a great introduction to coding for beginners and will help you to develop key programming skills for the future. In Get Coding! learn how to write code and then build your own website, app and game using the programming languages HTML, CSS and JavaScript. In Get Coding 2! discover how to design and build five interactive computer games using HTML and JavaScript. Create graphics and code simple artificial intelligence to play against. Then enjoy playing your finished games.
